## Automated Functional Tests of [The Internet App](https://the-internet.herokuapp.com/)

---

The tests were written using Selenium WebDriver, Java, and TestNG. The following sections of the app were covered:

* [Add/Remove Elements Test](https://the-internet.herokuapp.com/add_remove_elements)
* [Basic Auth Test](https://the-internet.herokuapp.com/basic_auth)
* [Checkboxes Test](https://the-internet.herokuapp.com/checkboxes)
* [Context Menu Test](https://the-internet.herokuapp.com/context_menu)
* [Drag and Drop Test](https://the-internet.herokuapp.com/drag_and_drop)
* [Dropdown Test](https://the-internet.herokuapp.com/dropdown)
* [Dynamic Loading Test](https://the-internet.herokuapp.com/dynamic_loading)
* [Entry Ad Test](https://the-internet.herokuapp.com/entry_ad)
* [Exit Intent Test](https://the-internet.herokuapp.com/exit_intent)
* [File Download Test](https://the-internet.herokuapp.com/download)
* [Frames Test](https://the-internet.herokuapp.com/frames)
* [Horizontal Slider Test](https://the-internet.herokuapp.com/horizontal_slider)
* [Inputs Test](https://the-internet.herokuapp.com/inputs)
* [Key Presses Test](https://the-internet.herokuapp.com/key_presses)
* [Notification Message Test](https://the-internet.herokuapp.com/notification_message)
